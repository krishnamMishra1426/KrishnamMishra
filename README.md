## 📂 GitHub Repository

## 🚀 Tech Stack

* Next.js
* React.js
* Tailwind CSS
* Nodemailer
* Vercel (Deployment)

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/krishnamMishra1426/KrishnamMishra.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Open http://localhost:3000 in your browser.
